# Sign Language Recognition System

## Overview

This project is developed to recognize American Sign Language (ASL) alphabet gestures using Machine Learning and Computer Vision techniques.

## Dataset

* ASL Alphabet Dataset
* Source: Kaggle
* Contains images of hand gestures representing A-Z alphabets.

## Technologies Used

* Python
* TensorFlow
* OpenCV
* NumPy
* Keras

## Working Process

1. ASL Alphabet Dataset was collected from Kaggle.
2. Images were preprocessed and used for model training.
3. A TensorFlow model was trained to recognize hand gestures.
4. Live hand gestures were captured using a webcam.
5. The trained model predicted the corresponding alphabet in real time.

## Features

* Real-time gesture recognition
* Live webcam capture
* ASL alphabet detection
* Machine Learning-based prediction

## Future Enhancements

* Sentence formation from gestures
* Voice output generation
* Mobile application integration
